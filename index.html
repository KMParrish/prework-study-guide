HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The picture's folder location is important for properly showcasing the picture when opening the HTML in a web browser.
  • Fowardslash marks the end of a sections content
  • ./ is used to reference a path relative from the current position

CSS

  • When using CSS to format a style, it is important to note that padding and boeders are not the same.
  • If using CSS, indicating VS Code to do so is necessary by using "style.css" within the desired directory.
  • Every aspect of the style can be manipulated using margin, width, or border commands (ex. margin: 40px auto;)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • "cd" is the command used used to change directories.
  • "ls" is useful for browsing directories that may be migrated to.
  • the command to pull and push is useful for cloning a file.

Javascript

  • Javascript reads from top to bottom, making it important to have all lines of code in a specific order.
  • Conditional statements permit the code to analyze for truths and perform a task.
  • Using Ctrl+Shift+I allows one to check the console on a webpage to double check the code for errors.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
